# Expense Tracker

A console-based Expense Tracker application developed in Python using Object-Oriented Programming (OOP). This project helps users record, organize, and analyze their daily expenses through a simple menu-driven interface.

---

## Features

- Add new expenses
- View all recorded expenses
- Generate monthly expense reports
- Generate category-wise expense reports
- Budget alert system
- Export expenses to a CSV file
- Menu-driven console interface

---

## Technologies Used

- Python 3
- Object-Oriented Programming (OOP)
- CSV Module

## How to Run

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Expense-Tracker.git
```

2. Open the project folder

```bash
cd Expense-Tracker
```

3. Run the program

```bash
python ExpenseTracker.py
```
